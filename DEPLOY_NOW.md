## 部署步驟

### 步驟 1: 推送到 GitHub
```bash

# 名稱: taiwan-landlord-system-ready
# 描述: 台灣房東系統 - 完整可上線版本

# 添加遠端倉庫
git remote add origin https://github.com/你的用戶名/taiwan-landlord-system-ready.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

### 步驟 2: 部署到 Vercel
1. 訪問 [Vercel](https://vercel.com)
2. 點擊 "New Project"
3. 選擇剛創建的 GitHub 倉庫
4. 點擊 "Import"
5. 保持默認設置，點擊 "Deploy"
6. 等待部署完成 (約 1-2 分鐘)
