## 📋 部署步驟

### 步驟 1: 進入 Vercel
1. 訪問 **[Vercel 官網](https://vercel.com)**
2. 點擊右上角 **"Sign Up"** 或 **"Log In"**
3. 使用 GitHub 帳號登入最方便

### 步驟 2: 創建新專案
1. 登入後點擊 **"New Project"**
2. 選擇 **"Import Git Repository"**
3. 授權 Vercel 訪問你的 GitHub 帳號

### 步驟 3: 選擇專案
1. 在倉庫列表中選擇 **"taiwan-landlord-v3"**
2. 點擊 **"Import"**

### 步驟 4: 部署設定
1. **專案名稱**: taiwan-landlord-v3 (自動生成)
2. **框架**: Next.js (自動偵測)
3. **根目錄**: 保持默認
4. **環境變數**: 不需要 (本地存儲)
5. 點擊 **"Deploy"**

## 🚨 故障排除

### 常見問題 1: 部署失敗
```
錯誤: Build failed
解決: 檢查 package.json 中的依賴版本
```

### 常見問題 2: 頁面空白
```
錯誤: 頁面加載但顯示空白
解決: 檢查瀏覽器控制台錯誤，清除緩存
```

### 常見問題 3: 數據不保存
```
錯誤: 刷新後數據丟失
解決: 檢查瀏覽器是否禁用了 localStorage
```
